# Visão Geral da Arquitetura

Esta seção descreve a stack de tecnologia e a arquitetura geral do projeto LibriShare.

## Stack de Tecnologias

O projeto é dividido em três componentes principais: Front-end, Back-end e Banco de Dados.

| Componente | Tecnologia | Descrição |
| :--- | :--- | :--- |
| **Front-end** | **v0.dev** | Interface do usuário prototipada e gerada com v0. |
| **Back-end** | **Java** / **Spring Boot** | API RESTful para lógica de negócios e gerenciamento de dados. |
| **Banco de Dados** | **PostgreSQL** | Banco de dados relacional para persistência dos dados. |
| **Infraestrutura** | **Docker** | Usado para conteinerizar a aplicação (back-end e banco de dados). |
