# Arquitetura Front-end

A interface do usuário (UI) do LibriShare foi desenvolvida e prototipada utilizando a plataforma **v0.dev**.

## Prototipação com v0

A tecnologia do v0 foi usada para gerar o design e os componentes de front-end com base em prompts de linguagem natural e refinamentos iterativos.

* **Link do Protótipo:** [v0.app/chat/libri-share-platform-design](https://v0.app/chat/libri-share-platform-design-msPJcnG8O41?b=b_qAcFelQOQEo&f=1)

## Telas Principais

O design do front-end, como visto no protótipo em PDF (`LibriShare.pdf`), foca em uma navegação limpa e uma interface escura (dark mode) para conforto de leitura.

* **Navegação Principal:** Um menu lateral fixo fornece acesso rápido às seções principais: Dashboard, Minha Biblioteca, Lendo Agora, Livros Lidos, Lista de Desejos, Empréstimos e LibriConnect.
* **Componentes Reativos:** A interface utiliza modais (pop-ups) para ações como "Adicionar Livro", "Registrar Empréstimo" e "Atualizar Progresso", mantendo o usuário no contexto da tela atual.
* **Design Responsivo:** O design foi planejado para ser adaptável, embora o protótipo foque na visualização de desktop.
